## 4. DCX A17 特殊設定 (B-series / C-series) ← **最大の差分**

A17フィールド (2bit) は "Main Subject for Specific Settings" を示し、A18フィールド (15bit) の解釈を変える。

> [!IMPORTANT]
> **IS-QZSS-DCX-003 §4.2.3.17 による制約**
>
> - **L-Alert・Local Gov** は A17 が **00 (B1) 固定**。B2〜B4 は仕様上これらのメッセージで送信されない。
> - B2〜B4 が実際に使われるのは **Outside Japan** メッセージのみ。
> - したがって、azaraC への B2〜B4 実装の実用的優先度は Outside Japan 対応が必要かどうかに依存する。

| A17値 | モード                    | azarashi       | azaraC        | 内容                                              | 対象メッセージ                      |
| ----- | ------------------------- | -------------- | ------------- | ------------------------------------------------- | ----------------------------------- |
| 0     | **B1** - 主楕円の精度向上 | ✅ C1-C4 解析  | ❌ **未実装** | `a18`(15bit)をC1-C4に分割して座標精度向上         | L-Alert / Local Gov / Outside Japan |
| 1     | **B2** - 危害中心位置     | ✅ C5-C6 解析  | ❌ **未実装** | 楕円中心からの相対ずれ量(±10度範囲)               | **Outside Japan のみ**              |
| 2     | **B3** - 第2楕円          | ✅ C7-C10 解析 | ❌ **未実装** | 第2楕円のシフト・縮尺・方位・ガイダンスライブラリ | **Outside Japan のみ**              |
| 3     | **B4** - 詳細ハザード情報 | ✅ D1-D36 解析 | ❌ **未実装** | ハザード種別ごとの詳細値 (30種以上のD-series)     | **Outside Japan のみ**              |

> [!CAUTION]
> **B4 (D-series)** はazarashiで最も充実した部分。azaraCでは`d.camf.a17`と`d.camf.a18`の生値は格納されるが、その中身の解釈（C1-C10, D1-D36）は一切行われていない。
>
> また D-series の詳細定義は IS-QZSS-DCX-003 ではなく **EWSS CAMF Specification (applicable document 2)** に委ねられているため、azarashi の D-series 実装の正確性は CAMF仕様書との照合が別途必要。

## 9. アーキテクチャ・設計思想の差

| 観点           | azarashi                       | azaraC                       |
| -------------- | ------------------------------ | ---------------------------- |
| 対象環境       | PC / サーバ / Raspberry Pi     | ESP32 / Arduino / 組み込み   |
| メモリ         | 無制限 (Python heap)           | Tagged Union で最小化        |
| 動的確保       | あり (list, dict, str)         | ❌ (ゼロ動的確保)            |
| 動作言語       | Python 3                       | C++11                        |
| エラー処理     | 例外 `QzssDcrDecoderException` | bool返却                     |
| 二言語ラベル   | ✅ ja/en 両方                  | ✅ ja/en 両方 (定義ファイル) |
| ラベル解決方法 | 実行時辞書引き                 | コンパイル時配列             |

## 10. 差分まとめ

### azaraC に**ない**機能 (azarashiにある)

| 優先度 | 機能                                   | 影響範囲                            | 備考                         |
| ------ | -------------------------------------- | ----------------------------------- | ---------------------------- |
| 🔴 高  | **DCX A17 B1 の C1-C4 解釈**           | L-Alert / Local Gov / Outside Japan | B1はL-Alert等でも発生する    |
| 🟡 中  | **DCX A17 B2/B3/B4 解釈**              | **Outside Japan のみ**              | 国内向けには仕様上不要       |
| 🟡 中  | **D-series 詳細ハザード情報 (D1-D36)** | B4モード (Outside Japan のみ)       | CAMF仕様書との別途照合が必要 |
| 🟡 中  | 南海トラフ複数ページ集約               | MT43 カテゴリ4                      |                              |
| 🟡 中  | 人間可読テキスト出力                   | デバッグ・表示用                    |                              |
| 🟢 低  | HEX入力                                | CLIツール用途                       |                              |
| 🟢 低  | ネットワーク入力 (`decode_stream`)     | サーバ用途                          |                              |

### azaraC が**優れている**点 (azarashiにない)

| 機能                                     | 詳細                                      |
| ---------------------------------------- | ----------------------------------------- |
| **unix_time 付き時刻構造体**             | GPS時刻ベース・安定したunixタイムスタンプ |
| **JSON出力**                             | Arduino/ESP32でそのままHTTP送信可能       |
| **ゼロ動的メモリ確保**                   | フラッシュ・SRAM制約環境に最適            |
| **GPSウィーク4mod法によるonset解決**     | `datetime.now()` 依存なし                 |
| **Alert Identity (更新/全解除照合キー)** | IS-QZSS-DCX-003 §4.2.3.1 準拠             |
